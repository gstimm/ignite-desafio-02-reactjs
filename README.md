<h1>Rocket Shoes</h1>

## 🚀 Como executar

- Clone o repositório
- Instale as dependências com `yarn`
- Inicie o servidor com `yarn start`

A aplicação pode ser acessada em [`localhost:3000`](http://localhost:3000).
